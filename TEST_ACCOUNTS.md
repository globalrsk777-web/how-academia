## How to Use

1. Navigate to the login page
2. Enter one of the email addresses above
3. Enter the corresponding password
4. Select the appropriate role from the dropdown
5. Click "Login"

## Note

- These accounts are automatically created when the application starts
- You can also create new accounts by logging in with any email/password combination (the system will auto-create the account)
- All accounts are stored in browser localStorage and will persist across sessions
- To clear all accounts, clear your browser's localStorage

## Additional Test Accounts

You can create additional test accounts by:
1. Logging in with a new email/password combination
2. The system will automatically create the account if it doesn't exist
3. Or use the Institution dashboard to add students/instructors
